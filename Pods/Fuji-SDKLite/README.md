# Fuji


## Installation

Fuji is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "Fuji"
```

There are three subspec for Fuji you can use in your iOS or OSX projects

```ruby
pod "Fuji/Hook"
```

```ruby
pod "Fuji/Utilities"
```

```ruby
pod "Fuji/vmShared"
```
